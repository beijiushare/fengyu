# 峰语
世传《峰语》者，路人甲所撰也。专记互联网人物峰哥与其门下众粉丝言行，采其语、录其事、存其神，成语录之体，为网史之稗钞。书成于公元二十一世纪上半叶，自互联网纪年五十七年始纂。一言一语，皆当时之风气；一笑一怒，俱此间之人情。

本项目依托于 Github Action bot，当符合要求的 issues 被提交，根据模版自动变更网站内容。网站托管在 [netlify.com](netlify.com)。

## 建议
要参与本项目完善《峰语》，您需要提交满足以下需求的 issue：
- 标题必须为：峰语
- 您可以发布其他标题的 issue，这不会影响网站内容
- 您需要为该 issue 打上 label ，这会将该 issue 归类到以此 label 命名的篇章中，没有打上 label 的 issue 会被过滤掉
- 内容为纯文本，尽管 issues 的编辑器对 markdown 支持得很好，但就本项目而言，markdown 格式是不必要的，如果发布了 markdown 这类带有格式的文本，网站不会渲染
- 对于具体内容，建议遵循以下格式
  - 对于峰哥的直接言论，建议的格式是：峰哥曰：「内容」
  - 对于峰哥回答问题，建议的格式是：问：「内容」对曰：「内容」
  - 对于峰哥粉丝等人的言行，建议的格式是：{花名}曰：「内容」，其中 {} 在实际编写中省略
  - 对于评论，遵循和上条同样的建议
  - 对于其他内容，暂时没有更多的建议
- 对于某个 issues 下添加的评论，为了避免杂乱，均不会纳入，但您可以通过手动编辑 issue 来添加有意思的评论，需要遵循以下格式
```
言论
---
评论一
---
评论二
```

对于以上的自动化流程，没有审核，因此请尽可能保证不发布无关内容，保证格式的统一。
